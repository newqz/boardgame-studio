## 游戏介绍

马尼拉是一款由丹麦游戏设计师Franz Gliasner设计的经典股票投资与航运经营桌游，于2004年由Zoch公司发行。

玩家扮演马尼拉湾的投资者和商人，通过股票、期货、保险、港口许可证等机制，在五家贸易公司的经营中获利。游戏融合了策略规划、风险管理与时机把握，每场游戏都是一次商业头脑的较量。

---

## 游戏目标

**最终目标**：成为最富有的投资者

- 某玩家达到 **15000** 菲律宾比索，或
- 完成 **完整局**（所有五艘船都完成航行）后，价值最高者获胜

## 游戏流程

每轮游戏分为 **五个阶段**，依次进行：

### 阶段一：放份额

每个玩家同时秘密决定本期购买的股票数量，然后同时展示。

**规则**：
- 玩家可购买任意公司的股票
- 每种公司最多10张股东证书
- 购买时将证书放在桌面中央（股东板外）

**操作**：
```
1. 玩家决定购买数量
2. 手握拳放在桌面中央
3. 倒数"一、二、三"后同时展示
4. 验证数量是否超过剩余股票
5. 超过则该玩家本期无法购买
```

### 阶段二：分红

银行家（起始玩家）抽取一张利润分配卡，根据卡上数字向各公司股东发放红利。

**利润分配规则**：
- 绳子公司：分配 3-5 的红利
- 石油公司：分配 4-6 的红利
- 大米公司：分配 5-7 的红利
- 香料公司：分配 6-8 的红利
- 丝绸公司：分配 7-9 的红利

**计算公式**：
```
股东红利 = 公司利润分配值 × 股东证书数量
```

### 阶段三：下注

玩家可对每艘船进行下注，猜测该船能否成功到港或中途沉没。

**下注选项**：
| 下注类型 | 赔率 | 条件 |
|----------|------|------|
| 港口赌注 | 3:1 | 船成功到港 |
| 沉船赌注 | 2:1 | 船中途沉没 |
| 失踪赌注 | 4:1 | 船未能到港且未沉没 |

**限制**：
- 每次最多下注200比索
- 沉船赌注只能在船只进入马尼拉湾后下注

### 阶段四：行船

从绳子公司开始，依次处理每艘船的航行。

**航行流程**：
```
1. 掷两颗骰子
2. 计算航行距离：两颗骰子之和 - 船只当前位置
3. 船只前进相应步数
4. 判定结果：
   - 到港 → 股东获利
   - 沉没（骰子相同） → 保险赔付
   - 失踪（超出轨道） → 无人获利
```

**特殊位置**：
| 位置 | 效果 |
|------|------|
| 第4格 | 平安无事 |
| 第5格 | 沉没！ |
| 第6格 | 平安无事 |
| 第7格 | 失踪！ |
| 第8格 | 平安无事 |

### 阶段五：结算

本轮所有结果结算完成后，将所有权证书和货币归还银行。

## 详细规则

### 股票购买

- 每种股票初始10张
- 购买后放入公开的股东板
- 股票可转让，但需支付10%手续费

### 期货交易

- 可购买期货合约
- 期货持有者可在任意时刻以挂牌价出售
- 期货价格随市场波动

### 保险机制

- 玩家可购买保险单
- 船只沉没时，投保玩家获得500比索赔偿
- 保险单不可转让

### 港口许可证

- 进入港口需持有许可证
- 每港5张，先到先得
- 无证船只不得进入港口

## 胜利条件

### 快速胜利（15000比索）

任何玩家在游戏过程中个人资产达到或超过15000比索时，游戏立即结束，该玩家获胜。

### 完整局胜利

当所有五艘船都完成航行后游戏结束：
1. 计算每位玩家的总资产
2. 包括：现金 + 股票市值 + 期货价值 + 其他资产
3. 资产最高者获胜

**资产计算**：
```
总资产 = 现金 + (股票数量 × 当前股价) + 期货市值 + 保险赔偿
```

## 游戏技巧

### 新手建议

1. **分散投资**：不要把所有资金放在一家公司
2. **关注分红**：利润分配是稳定收入来源
3. **谨慎下注**：沉船和失踪是小概率事件
4. **保留现金**：留足资金应对机会

### 进阶策略

1. **期货套利**：低买高卖赚取差价
2. **联动投资**：相关公司同时持有降低风险
3. **时机把握**：在低价时买入，高价时卖出
4. **风险对冲**：同时购买股票和保险

## 常见问题

**Q：游戏可以几个人玩？**
A：2-5人最佳

**Q：游戏时长大约多久？**
A：45-90分钟

**Q：可以中途加入吗？**
A：不建议，游戏需要连续进行

**Q：是否有扩展包？**
A：暂无官方扩展包
